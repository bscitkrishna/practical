2. Classification algorithm
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.linear_model import LogisticRegression
from sklearn.metrics import accuracy_score, confusion_matrix, classification_report
iris = load_iris()
X = iris.data
y = iris.target
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.3, random_state=42)
model = LogisticRegression(max_iter=200)
model.fit(X_train, y_train)
y_pred = model.predict(X_test)
print("Accuracy:", accuracy_score(y_test, y_pred))
print("\nConfusion Matrix:\n", confusion_matrix(y_test, y_pred))
print("\nClassification Report:\n", classification_report(y_test, y_pred))

This program uses the Iris dataset from scikit-learn to perform a classification task using the Logistic Regression algorithm, which is a supervised machine learning method. The dataset contains 150 samples of iris flowers with four features (sepal length, sepal width, petal length, and petal width) and three target classes (Setosa, Versicolor, Virginica). The data is divided into training (70%) and testing (30%) sets using train_test_split, where random_state=42 ensures consistent results. The model is trained using model.fit() on the training data and then used to predict test data with model.predict(). The performance is evaluated using accuracy score (overall correctness), confusion matrix (detailed class-wise prediction results), and classification report (precision, recall, and F1-score), which together measure how well the model classifies the flower species.

This program implements the K-Means clustering algorithm, which is an unsupervised machine learning technique used to group similar data points into clusters. First, a small 2D dataset is created using NumPy. Then, the KMeans model from sklearn.cluster is initialized with 3 clusters and fitted to the data using .fit(X). The algorithm automatically assigns each data point to the nearest cluster center and calculates the centroids (mean position of each cluster). The labels show which cluster each point belongs to, and the centroids represent the center of each group. Finally, Matplotlib is used to visualize the clustered data points and their centroids, helping us clearly see how the data is divided into three groups.

# Step 1: Import Libraries
import matplotlib.pyplot as plt
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.tree import DecisionTreeClassifier, plot_tree
from sklearn.metrics import accuracy_score, confusion_matrix, classification_report

# Step 2: Load Dataset
iris = load_iris()
X = iris.data
y = iris.target

# Step 3: Split Data (70% Training, 30% Testing)
X_train, X_test, y_train, y_test = train_test_split(
    X, y, test_size=0.3, random_state=42)

# Step 4: Create and Train Model
model = DecisionTreeClassifier(criterion='gini', random_state=42)
model.fit(X_train, y_train)

# Step 5: Make Predictions
y_pred = model.predict(X_test)

# Step 6: Evaluate Model
print("Accuracy:", accuracy_score(y_test, y_pred))
print("\nConfusion Matrix:\n", confusion_matrix(y_test, y_pred))
print("\nClassification Report:\n", classification_report(y_test, y_pred))

# Step 7: Visualize Decision Tree
plt.figure(figsize=(12,8))
plot_tree(model, feature_names=iris.feature_names,
          class_names=iris.target_names,
          filled=True)
plt.title("Decision Tree Visualization")
plt.show()
This program implements a Decision Tree classifier using the Iris dataset. The dataset is divided into training and testing sets, and the model is trained using the Gini index criterion. The trained model predicts flower species based on input features. The performance is evaluated using accuracy, confusion matrix, and classification report. Finally, the decision tree is visualized to understand how the model makes classification decisions.
